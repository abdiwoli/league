# Football League Manager

A robust full-stack application for managing football leagues, teams, matches, and statistics.

## Features
- **League Management**: Create and manage seasons.
- **Team Management**: Register and manage teams with logos.
- **Match Scheduling**: Auto-generate schedules (Round Robin, etc.) and custom fixtures.
- **Live Updates**: Real-time table and statistics updates.
- **Player Stats**: Track goals, assists, and cards.

## Tech Stack
- **Frontend**: React, TypeScript, Tailwind CSS, TanStack Query
- **Backend**: Node.js, Express, Prisma ORM
- **Database**: PostgreSQL (Neon)

## Deployment
- Frontend: Vercel
- Backend: Render
